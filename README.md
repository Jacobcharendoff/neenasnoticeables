# Neena's Noticeables

Landing page for [neenasnoticeables.ca](https://neenasnoticeables.ca) — handmade gifts by Neena, a blind maker and entrepreneur in Toronto. Made by hand, chosen by touch.

## Stack

Single static `index.html`. No build step, no dependencies. Fonts from Google Fonts (Baloo 2, Caveat, Atkinson Hyperlegible).

## Deploy

Any static host works. Recommended: Vercel or Netlify — import this repo, no config needed, then point the `neenasnoticeables.ca` DNS at it.

## Accessibility

Built screen-reader-first: semantic landmarks, strict heading hierarchy, skip link, WCAG AA+ contrast, visible focus states, and body text in Atkinson Hyperlegible (the Braille Institute's low-vision typeface).
